## Next: Make Your Video (60-90 min total)

### 1. Record Narration (10 min)
```
Open Audacity (free)
Read from: video-narration.md
Save as: narration.mp3
```

### 2. Create Slides (30 min)
```
Open Canva or Figma (free)
Reference: slide-deck.md
Add diagrams from: video-graphs.md
Export as: slide_1.png, slide_2.png, etc.
```

### 3. Generate AI Video (5 min)
```
Go to: https://pika.art (free credits)
Upload: Slides + narration
Generate: video.mp4
```

### 4. Edit & Upload (15 min)
```
Open DaVinci Resolve (free)
Import: video.mp4
Follow: video-editing-guide.md
Export to YouTube
```
