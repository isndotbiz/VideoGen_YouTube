### HOUR 3: Animation & Assembly

**Step 1: Create Animated Clips (60 minutes)**

**Option A: SpeedSketch for Diagrams (Recommended)**
1. Go to https://www.speedsketch.co.uk/app
2. For each of your 8+ images:
   - Upload image (JPG/PNG)
   - Select drawing style (random/spiral/diagonal)
   - Adjust speed and duration (match narration timing)
   - Download as MP4
3. Save as: `animation_01.mp4`, `animation_02.mp4`, etc.
4. Time investment: ~45 minutes for 8 animations

**Option B: Snappify for Code Sections (if you have code comparisons)**
1. Go to https://snappify.com
2. Paste before/after code
3. Generate animated comparison
4. Download as MP4

**Step 2: Assemble in Canva (45 minutes)**

1. Go to https://www.canva.com/video-editor/
2. Create new video project (1920x1080)
3. Import all `animation_XX.mp4` files
4. Arrange in timeline in order
5. Add your `narration.mp3` as background track
6. Add text overlays between scenes (optional):
   - Use Canva's built-in text animations
   - "Typewriter" effect for key points
   - Fade in/out for transitions
7. Add background music from Canva's free library (keep at 20-30% volume)
8. Review timing - should be ~17 minutes total
9. Export as 1080p MP4

**Result:** `video_final.mp4`
