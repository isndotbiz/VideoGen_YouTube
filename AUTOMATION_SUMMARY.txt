COMPLETE AUTOMATED VIDEO GENERATION SYSTEM
================================================================================

WHAT YOU HAVE:
- setup_apis.py: Interactive API key setup & testing
- video_pipeline.py: Main orchestration engine (all 5 phases)
- comfyui_batch_generator.py: Local Flux Turbo batch image generator
- requirements.txt: All Python dependencies
- .env.example: Environment template

QUICK START (15 minutes):
1. pip install -r requirements.txt
2. cp .env.example .env
3. python setup_apis.py (follow prompts to add API keys)
4. python video_pipeline.py --help

WORKFLOW:
Article → [Images] → [Narration] → [Video Assembly] → [YouTube Upload]
         ComfyUI   ElevenLabs    Shotstack         YouTube API

PHASE BREAKDOWN:

PHASE 1: IMAGE GENERATION (30 min)
- Tool: ComfyUI local (free) OR FAL.ai (fallback)
- Generates: 21 images at 1920x1080
- Speed: 3-5 seconds each with Flux Turbo + LoRA
- Cost: $0.03 (if using FAL.ai)

PHASE 2: NARRATION (5 min)
- Tool: ElevenLabs Text-to-Speech API
- Generates: MP3 narration from script
- Cost: ~$0.02

PHASE 3: VIDEO ASSEMBLY (30-60 min)
- Tool: Shotstack Video Assembly API
- Combines: Images + narration → MP4
- Cost: $3.40 for 17-minute video

PHASE 4: YOUTUBE UPLOAD (5 min)
- Tool: YouTube Data API v3
- Publishes: Video with metadata
- Cost: FREE (10k units/day quota)

TOTAL COST PER VIDEO: ~$3.45
(Or $0 if using ComfyUI locally for images)

API KEY SETUP:
1. FAL.ai: https://fal.ai/dashboard/keys
2. ElevenLabs: https://elevenlabs.io/app/settings/api-keys
3. Shotstack: https://dashboard.shotstack.io/settings/api-key
4. YouTube: https://console.cloud.google.com (OAuth)
5. ComfyUI: Local (http://localhost:8188)

RUNNING THE PIPELINE:

Full pipeline (all phases):
  python video_pipeline.py \
    --prompts prompts.json \
    --script "VIDEO_SCRIPTS_ALL_VARIATIONS.md" \
    --title "Claude Code vs Codex"

Just images:
  python comfyui_batch_generator.py --prompts prompts.json

Just narration:
  python video_pipeline.py --phase narration --script "script.txt"

Just assembly:
  python video_pipeline.py --phase assembly \
    --images ./generated_images --audio narration.mp3

Just YouTube upload:
  python video_pipeline.py --phase upload --video video.mp4

ENVIRONMENT SETUP:
1. Copy .env.example to .env
2. Fill in 5 API keys (see above)
3. Save .env in project directory
4. IMPORTANT: Add .env to .gitignore (never commit!)

TROUBLESHOOTING:
Problem: "API key invalid"
Solution: Re-run python setup_apis.py, copy key exactly

Problem: "ComfyUI not reachable"
Solution: Ensure running (python main.py) and port 8188 accessible

Problem: "Video assembly timeout"
Solution: Normal for 17+ min videos, takes 1-2 min per minute

Problem: "YouTube upload fails"
Solution: First upload requires web browser auth, check OAuth setup
