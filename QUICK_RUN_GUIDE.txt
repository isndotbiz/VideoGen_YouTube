WHAT YOU NEED: 2 minutes setup, 15 minutes to create video

================================================================================

SETUP (One time)

1. Set environment variables:

   Windows Command Prompt:
     set ELEVENLABS_API_KEY=your_key_here
     set ASSEMBLYAI_API_KEY=your_key_here

   Windows PowerShell:
     $env:ELEVENLABS_API_KEY = "your_key_here"
     $env:ASSEMBLYAI_API_KEY = "your_key_here"

   Linux/Mac:
     export ELEVENLABS_API_KEY="your_key_here"
     export ASSEMBLYAI_API_KEY="your_key_here"

2. Verify you have:
   - ffmpeg installed
   - ffprobe installed
   - Python 3.7+
   - elevenlabs library (pip install elevenlabs)

================================================================================

RUN THE SCRIPT

cd /d/workspace/VideoGen_YouTube

python create_full_30sec_test_video.py

That's it. Watch the console output for progress.

================================================================================

WHAT HAPPENS (Step by step)

[1] Generate narration (ElevenLabs TTS)
    Reads: ELEVENLABS_CLEAN_NARRATION_30SEC.txt
    Creates: narration_test_30sec.mp3 (2.3 MB)

[2] Prepare animations
    Uses: Existing animation files (if available)
    Fallback: Black background

[3] Get background music
    Looks for: pexels_ambient_bgm.mp3
    Volume: 40% (underneath narration)

[4] Generate subtitles
    Uses: AssemblyAI speech-to-text
    Creates: subtitles_test_30sec.srt

[5] Create Shotstack config
    Format: JSON configuration
    Format: 1920x1080, 30 seconds, H.264

[6] Compose video
    Tool: FFmpeg (free, fast)
    Creates: video_test_30sec.mp4 (4-5 MB)

[7] Add YouTube SEO metadata
    Title: "The 8 Best Free AI Tools - 30 Second Test"
    Tags: AI tools, ChatGPT, Midjourney, etc.
    Privacy: PRIVATE (for testing)

[8] Verify quality
    Checks: File size, duration, codecs
    Status: Ready for YouTube

[9] YouTube upload (optional)
    Shows: Command to upload

Total time: ~15-20 minutes
