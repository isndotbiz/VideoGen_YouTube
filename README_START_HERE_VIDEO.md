- Venn diagram (overlap)
- Before/After code comparison
- Handshake partnership

### Step 2: Narration (15 minutes)
1. Go to **https://elevenlabs.io**
2. Copy text from: **VIDEO_SCRIPTS_ALL_VARIATIONS.md** ‚Üí SCRIPT 1 (Full 17-min version)
3. Paste into ElevenLabs
4. Click "Generate"
5. Download as `narration.mp3`

**Why ElevenLabs:**
- Free tier includes thousands of characters
- Professional-quality voices
- Takes 5-10 minutes
- Downloads instantly

### Step 3: Animate (45 minutes)
1. Go to **https://www.speedsketch.co.uk/app**
2. For each of your 8-10 PNG images:
   - Upload image
   - Select drawing style (random/spiral)
   - Adjust speed to match narration
   - Download as MP4
3. Save as: `animation_01.mp4`, `animation_02.mp4`, etc.

**Why SpeedSketch:**
- 100% free
- No watermark
- Whiteboard animation effect (looks professional)
- Downloads immediately

### Step 4: Assemble (45 minutes)
1. Go to **https://www.canva.com/video-editor/**
2. Create new video (1920x1080)
3. Import all `animation_XX.mp4` files in order
4. Add `narration.mp3` as audio track
5. (Optional) Add background music from Canva's free library
6. Export as 1080p MP4
7. Save as `video_final.mp4`

**Why Canva:**
- 30-minute max duration (perfect for your 17-min video)
- Free tier includes 1000s of elements
- No watermark with free assets
- Easy timeline editing

### Step 5: Upload (5 minutes)
1. Go to **https://youtube.com**
2. Upload `video_final.mp4`
3. Add title: "Claude Code vs Codex: Why I Use Both (And You Should Too)"
4. Paste description from MASTER_VIDEO_CREATION_PLAN.md
5. Publish
